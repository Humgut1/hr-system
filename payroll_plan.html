📜 기능 3 — 급여 이력 (Salary History)
현재 (문제)
employee_salary 테이블에 현재 급여만 저장
급여 수정 시 이전 값 덮어씀 → 이력 없어짐
언제, 얼마였는지 확인 불가
인상 근거·사유 기록 불가
개선 후
salary_history 테이블 — 변경 시마다 자동 INSERT
변경일·변경 전후 금액·변경자·사유 저장
직원 프로필 급여 탭에서 이력 타임라인 확인
Admin도 전 직원 이력 조회 가능
신규 DB 테이블 — salary_history
CREATE TABLE salary_history ( id INTEGER PRIMARY KEY AUTOINCREMENT, user_id INTEGER NOT NULL REFERENCES users(id), changed_at DATETIME DEFAULT CURRENT_TIMESTAMP, changed_by INTEGER REFERENCES users(id), -- 변경한 Admin user_id old_base_salary INTEGER DEFAULT 0, new_base_salary INTEGER DEFAULT 0, old_meal INTEGER DEFAULT 0, new_meal INTEGER DEFAULT 0, old_transport INTEGER DEFAULT 0, new_transport INTEGER DEFAULT 0, reason TEXT -- 변경 사유 (선택) );

📊 기능 4 — 일괄 인상 (Bulk Salary Raise)
현재 (문제)

총 인건비 증가액 미리 계산
salary_history 자동 기록
