## v0.35.0 — 랜딩 페이지 콘텐츠 전면 개편

### 1인 HR 담당자 타겟 메시지로 전환
- 히어로: "1인 HR팀도 대기업처럼" — 혼자서 200명을 관리하는 현실에 공감하는 카피로 교체
- 타사 대비 최대 7배 저렴한 가격을 전면에 강조
- 히어로 플로팅 카드: 오늘 처리할 업무 목록 (휴가 승인 / 전자서명 / 증명서 발급) 실무 반영

### 경쟁사 비교표 신규 추가
- TalentCore vs 국내 HR SaaS A vs 글로벌 HR SaaS B 기능·가격 직접 비교
- 7개 기능 항목별 지원 여부 (퇴직금 자동계산, 한국 근로기준법 준수 등 차별점 강조)
- 50명 기준 월 비용 직접 비교

### 비용 계산기 (Cost Simulator) 신규
- 슬라이더로 직원 수(5~200명) 조절 시 월 비용 실시간 비교
- 타사 대비 연간 절약 금액 자동 계산

### 요금제 전면 수정 — 실제 요금 반영
- 기존 허구의 3티어(₩0 / ₩9,900 / 맞춤) → **단일 플랜 ₩1,000/인/월** 로 교체
- High-Water Mark 과금 방식 명확히 설명
- 전체 포함 기능 목록 4개 카테고리 정리 (급여노무 / 근태휴가 / 계약증명 / 성과채용)
