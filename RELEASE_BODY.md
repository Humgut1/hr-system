## v0.29.0 — 휴일근로 수당 자동화 + 공휴일 관리

### 공휴일 DB 연동
- `public_holidays` 테이블 — 연도별 공휴일 저장
- 2026년 한국 법정 공휴일 15일 자동 시드 (신정·설날·삼일절·어린이날·부처님오신날·현충일·광복절·추석·개천절·한글날·크리스마스 등)
- Admin > **Public Holidays** 메뉴에서 연도별 조회, 추가, 삭제

### 휴일근로 수당 자동 계산 (근로기준법 §56②)
- 직원이 공휴일에 체크아웃하면 자동으로 `holiday_min` 기록
- 급여 생성 시 휴일근로 수당 자동 반영
  - 8시간 이내: 통상임금 × **1.5배**
  - 8시간 초과: 통상임금 × **2.0배**
  - 야간 중복 시 야간가산 추가 적용
