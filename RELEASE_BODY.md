## v0.31.0 — 전자계약 개편 + 1:1 미팅 제거

### 전자계약 기능 전면 개편
- 근로계약서 / NDA / 수습근로계약서 / 프리랜서 계약서 4종 양식 재작성 (고용노동부 표준 양식 기준)
- 근로계약서에 **휴일 조항 추가** (근로기준법 제55조 — 법정 필수 항목)
- 계약서 발행 시 `{{employee_name}}`, `{{salary}}`, `{{department}}` 등 변수 자동 치환
- 계약서 보기 페이지에 문서 용지 스타일 적용 (인쇄 최적화)
- 템플릿 생성 / 계약서 발행 페이지에 편집·미리보기 탭 추가

### 버그 수정
- `contract_issue` 직원 쿼리 컬럼명 오류 수정 (`dept_id` → `department_id`)
- 계약서 목록 템플릿 쿼리 `created_at` 누락 수정 (500 오류 해결)

### 기능 제거
- **1:1 미팅** 기능 제거 (라우트 3개, 템플릿 3개, 사이드바 링크 2개)
