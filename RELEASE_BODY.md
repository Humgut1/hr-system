## v0.36.0 — 랜딩 페이지 전면 재설계 (토스 스타일)

### 히어로 애니메이션 교체
- 기존 4개 플로팅 카드 → 단일 대시보드 카드로 교체
- 4가지 HR 업무(급여 생성 / 계약서 서명 / 연차 승인 / 증명서 발급)가 순차 완료되는 애니메이션
- 모든 업무 완료 시 "오늘 업무 모두 처리했어요" 배너 + 이번 달 자동처리 건수 카운터 표시
- 4초 주기 반복 루프

### 전체 카피 ~해요 체 전환 (토스 스타일)
- 헤드라인: "직원 100명, HR 1명이어도 괜찮아요."
- 부제: "급여 계산, 연차 관리, 계약서 발행, 채용까지. 매달 반복되던 그 일들을 TalentCore가 대신해요."
- 최종 CTA: "이번 달 급여일, 야근하지 않아도 돼요."
- 전체 섹션(기능/스토리/비교/후기/FAQ) ~해요 체 통일
