## v0.37.0 — 랜딩 페이지 재설계 (Claude Design 참고)

### 디자인 방향 전환
- Claude Design 번들과 기존 버전을 비교 분석 후 장점만 선별 통합
- "한 섹션 = 한 메시지" 원칙 적용 — 섹션마다 `min-height: 100vh` 구조
- Pretendard 폰트 적용 (한국어 가독성 향상)
- 메인 헤드라인 크기 축소 (기존 대비 약 30% 감소)

### 히어로 섹션
- 기존 대시보드 태스크 애니메이션 → 폰 목업 + 플로팅 알림 카드로 교체
- 폰 화면: 급여명세서 / 잔여 연차 / 평가 마감 실무 정보 표시
- 플로팅 카드: 휴가 승인 / 계약서 서명 / 급여 계산 완료

### 신규 섹션
- **숫자 스트립**: 도입 기업 / 관리자 만족도 / 타사 대비 절감 비용 — 카운트업 애니메이션
- **다크 인용 섹션**: 실제 사용자 후기 + 핵심 지표 4개 (어두운 배경)

### 기능 섹션 전면 교체
- 벤토 그리드 → 교차형 좌우 배치 (텍스트 + 실제 UI 목업)
- 급여명세서 다크 카드 목업 (실제 항목별 계산 표시)
- 연차 승인 리스트 목업 (이름 / 기간 / 상태)
- 전자계약 현황 목업 (서명 완료 / 대기 / 초안 상태)
- 채용 칸반 파이프라인 목업 (서류→면접→최종)

### 스크롤 애니메이션 개선
- IntersectionObserver 기반 reveal 애니메이션
- 지연(delay) 클래스로 순차 등장
