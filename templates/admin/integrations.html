.env 설정 가이드

프로젝트 루트의 .env 파일에 아래 값을 추가하면 데모 모드에서 실제 연동으로 자동 전환됩니다.

# Slack
SLACK_BOT_TOKEN=xoxb-...
SLACK_ADMIN_TOKEN=xoxp-...
SLACK_WORKSPACE_ID=T...

# Jira + Confluence (같은 토큰 사용)
JIRA_BASE_URL=https://your-domain.atlassian.net
JIRA_EMAIL=your@email.com
JIRA_API_TOKEN=...
JIRA_PROJECT_KEY=HR

# Confluence
CONFLUENCE_BASE_URL=https://your-domain.atlassian.net/wiki
CONFLUENCE_SPACE_KEY=TEAM
Atlassian API 토큰은 id.atlassian.com → 보안 → API 토큰에서 발급합니다. Jira와 Confluence가 같은 계정이면 토큰 하나로 둘 다 사용 가능합니다.
