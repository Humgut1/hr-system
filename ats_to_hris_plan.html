필드 오퍼에 있는 값 직원 레코드로 누가 채우나
이름 지원서에서 users.name 자동
이메일 지원서에서 users.email 자동
연락처 지원서에서 users.phone 자동
부서 요청서(Requisition)에서 users.department_id 자동
직급 요청서에서 users.position_id 자동
기본급 오퍼 base_salary payslips 초기 등록 자동
입사일 오퍼 hire_date users.hire_date 자동
근무 스케줄 — user_schedule_assignments HR 배정
버디 — users.buddy_id HR 배정
주소 — users.address 본인 입력
비상연락처 — users.emergency_* 본인 입력
계좌번호 — (추후 급여계좌 테이블) 본인 입력
온보딩 대시보드 — 입사자 화면 목업
