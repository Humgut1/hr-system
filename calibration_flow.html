3
Phase 3 · 자동 집계 + 등급 산출
시스템이 점수를 자동 계산
캘리브레이션 페이지 접속 시 실시간 계산 · 저장하지 않음
-- 종합 점수 계산 (있는 평가만 포함) overall = avg(self_avg, peer_avg, mgr_avg) -- 등급 자동 산출 IF overall >= 4.5 → S (탁월) IF overall >= 3.5 → A (우수) IF overall >= 2.5 → B (양호) IF overall >= 1.5 → C (개선 필요) ELSE → D (미흡) -- 이상 감지 IF |self_avg - mgr_avg| >= 1.5 → ⚠ 노란 배경 + 경고 배지
⚠ 이상 감지 케이스
· 자기평가 5.0 / 매니저평가 2.8 → "자기평가가 매니저평가보다 현저히 높음" (자기인식 과잉)
· 자기평가 2.0 / 매니저평가 4.5 → "매니저평가가 자기평가보다 현저히 높음" (겸손한 자기평가)
→ HR이 해당 직원 매니저와 별도 논의 필요
등급점수 범위의미비율 가이드 (강제 아님)
S4.5 이상탁월 — 기대를 크게 초과약 5~10%
A3.5 ~ 4.49우수 — 기대를 초과약 20~25%
B2.5 ~ 3.49양호 — 기대에 부합약 50~60%
C1.5 ~ 2.49개선 필요약 10~15%
D1.49 이하미흡약 3~5%

📌 전체 흐름 한 줄 요약

📅 주기 개설 → ✍️ 자기평가 + ⭐ 동료평가 + 👔 매니저평가 → ⚖️ HR 등급 확정 → 📤 직원 공개
· 강제 등급 분포 없음 — 점수 기반으로 자동 산출하되, HR이 최종 조정
· 이상 감지 (자기평가 vs 매니저평가 1.5점 이상 차이) 직원은 별도 검토 권장
· 확정 후 공개 전까지 직원에게 결과 비공개 — HR이 준비되면 "공개" 버튼 클릭
· 공개 시 인앱 알림 자동 발송 → 직원이 성과 페이지에서 결과 확인
