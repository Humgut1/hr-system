기능 개요

HR Admin 전용 통합 분석 대시보드입니다. 6개 탭으로 구성되며 실시간 인력 현황, 보상 형평성, 퇴직자 패턴 분석, 52시간 위반 모니터링, 자유 형식 HR 데이터 마법사를 제공합니다. Excel 내보내기(7종)도 이 페이지에서 실행됩니다.

② Pay Equity (보상 형평성)

· 전 직원 Compa-Ratio 산포도 (이상치 하이라이트)
· 부서별 평균 Compa-Ratio 비교
· 밴드 하단(80% 미만) / 상단(120% 초과) 위치 직원 목록
· 이상치 직원 직접 링크 (직원 프로필로 이동)
Compa-Ratio 이상치(80% 미만 또는 120% 초과)는 보상 불형평의 신호입니다. 정기적으로 확인하고 Merit 검토 시 우선 반영하세요.

③ 퇴직자 분석

📌 퇴직자 분석 활용 방법

최근 24개월 완료된 퇴직 건이 자동 집계됩니다.

KPI 확인

아쉬운 퇴직 비율이 높으면 보상·문화 개선이 필요. 재채용 가능 비율은 채용 전략에 활용

매니저별 이탈 Top 5

특정 매니저 팀에서 이탈이 집중되면 매니저십 이슈로 판단. 1:1 면담·코칭 검토

퇴직자 분석 데이터가 부족하면 퇴직 승인 시 아쉬운 퇴직 여부, 재채용 가능 여부, 실제 이탈 원인을 반드시 입력하도록 HR 프로세스를 수립하세요.
