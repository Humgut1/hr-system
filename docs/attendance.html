근태 관리

출퇴근 체크인·아웃, 휴가 신청·승인, 연장근로 관리, 52시간 모니터링

기능 개요

근태 관리 모듈은 직원의 출퇴근 기록을 자동 집계하고 연장·야간·휴일 수당을 근로기준법 기준으로 자동 계산합니다. 휴가는 13종 유형을 지원하며 매니저·HR 2단계 승인 워크플로우를 따릅니다. 연장근로(OT) 사전·사후 신청도 가능하며 승인된 OT는 다음 달 급여에 자동 반영됩니다.

출퇴근 체크인·아웃 흐름

수당 계산 기준: 연장근로 통상시급 × 1.5배, 야간(22시~06시) +0.5배, 휴일 +1배 (근로기준법 §56)

휴가 신청·승인 흐름

연차 신청 시 잔여일수 부족, 중복 기간이 자동 검사됩니다. 고정일수 유형(출산휴가 90일 등)은 날짜가 자동 설정됩니다.

역할별 사용법

📌 출퇴근 체크인·아웃

근태 메뉴 접속

사이드바 근태 클릭 → 오늘의 출퇴근 현황 확인

체크인

출근 시 체크인 버튼 클릭. 지각 여부가 자동 판정됩니다.

체크아웃

퇴근 시 체크아웃 클릭. 근무시간·연장수당이 즉시 계산됩니다.

📌 휴가 신청

휴가 탭 선택

근태 페이지 → 휴가 탭 → + 신청

유형 선택

카드 형식으로 휴가 유형 선택. 법령 근거와 남은 일수가 표시됩니다.

기간 입력 후 제출

시작일·종료일 입력. 고정 일수 유형은 자동 설정됩니다.

승인 대기

매니저 검토 → HR 승인 완료 시 알림이 옵니다.

근태 관리 홈 — 출퇴근 현황 및 휴가 신청 탭

📌 팀원 휴가·OT 승인

승인 탭 접속

근태 페이지 → 승인 탭에서 팀원 신청 목록 확인

내용 검토 후 처리

신청 내용 확인 후 승인 또는 반려 (반려 시 사유 필수)

HR로 이관

매니저 승인 완료 시 HR에게 자동으로 최종 승인 요청이 전달됩니다.

본인 소속 팀원만 승인할 수 있습니다. 타 부서 신청은 표시되지 않습니다.

📌 HR 최종 승인 처리

대시보드 Inbox 확인

어드민 대시보드 → Inbox에서 HR 승인 대기 건 목록 확인

최종 승인

근태 → 승인 탭에서 일괄 또는 개별 처리

연차 이월 (연도 교체 시)

근태 → OT 탭 → 연차 이월 처리 버튼으로 전 직원 잔여연차를 다음 연도로 이월

📌 52시간 모니터링

분석·보고서 접속

분석·보고서 → 52h 모니터링 탭

위반·경고 직원 확인

위반(52h 초과) 및 경고(48h 이상) 직원 목록과 주별 추이 차트 확인

체크아웃 시 자동 알림

52h 초과 감지 시 해당 직원의 매니저와 HR에게 인앱 알림이 즉시 발송됩니다.

주의사항
주 52시간 초과 근무는 근로기준법 위반입니다. 시스템이 자동 감지하여 알림을 보내지만, 실제 업무 조정은 매니저가 직접 해야 합니다.
체크인과 체크아웃이 동일 시각인 경우 0분 처리됩니다. 실수로 동시에 누른 경우 HR에게 수동 수정을 요청하세요.
반차(오전/오후)는 0.5일로 차감됩니다. 연차 잔여일수가 0.5일 미만인 경우 신청이 차단됩니다.
