TalentCore 사용자 가이드

200명 규모 스타트업을 위한 통합 HR 관리 시스템입니다.
채용부터 퇴직까지 전체 직원 라이프사이클을 하나의 플랫폼에서 관리합니다.

라이브 데모 열기
역할(Role) 안내

TalentCore는 4가지 역할로 접근 권한이 구분됩니다. 각 역할은 볼 수 있는 메뉴와 수행할 수 있는 작업이 다릅니다.

🛡️

HR Admin

  • 전체 직원 관리
  • 급여 생성·수정
  • 성과 캘리브레이션
  • 채용 최종 승인
  • 분석·보고서 전체
  • 시스템 설정
👔

Manager

  • 소속 팀원 근태 승인
  • 팀원 성과 평가
  • 다면평가 배정
  • 후계자 계획 조회
  • 채용 요청서 작성
  • 인사발령 기안
👤

Employee

  • 출퇴근 체크인·아웃
  • 휴가·OT 신청
  • 목표 설정·자기평가
  • 다면평가 작성
  • 급여명세서 열람
  • 증명서·계약서 조회
🎯

Recruiter

  • 채용 공고 관리
  • 지원자 파이프라인
  • 면접 배정
  • 오퍼 발행
  • 채용 대시보드
기능 모듈 가이드

조직 관리

직원 등록·수정, 부서·직급 체계, 인사발령, 조직도, 스킬·자격증, 퇴직 처리

근태 관리

출퇴근 체크인·아웃, 휴가 신청·승인, 연장근로 관리, 유연근무, 52시간 모니터링

성과 관리

목표 설정, 자기·매니저·다면평가, 캘리브레이션, Talent Card, 후계자 계획

보상 관리

월급여 생성, Compa-Ratio, Merit Matrix, 성과 연동 급여 검토, 일괄 인상

채용 관리

채용 요청서, 공고 관리, 지원자 파이프라인, 면접·오퍼, 합격자→직원 전환

내 문서

급여명세서 열람, 재직·경력·소득 증명서 발급, 전자계약서 서명·조회

분석·보고서

인력 분석, Pay Equity, 퇴직자 분석, 52h 모니터링, HR 데이터 마법사
시작하기

로그인 방법

시스템 URL로 접속하면 로그인 화면이 표시됩니다. HR Admin이 발급한 계정(이메일 + 임시 비밀번호)으로 로그인합니다.

게스트 계정(guest@talentcore.com / guest1234!)으로 읽기 전용 체험이 가능합니다.
처음 HR Admin으로 로그인하면 회사 설정 마법사(5단계)가 자동 실행됩니다. 근무제도·휴가 정책·급여 정책을 입력하면 시스템이 완전히 활성화됩니다.
