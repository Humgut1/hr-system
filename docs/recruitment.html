채용 관리

채용 요청서, 공고 관리, 지원자 파이프라인, 면접·오퍼, 합격자 직원 전환

기능 개요

채용 관리 ATS(Applicant Tracking System)는 채용 요청서 승인부터 공고 게시, 지원자 파이프라인, 면접 관리, 오퍼 발행, 합격자의 직원 자동 전환까지 전체 채용 라이프사이클을 관리합니다.

채용 전체 흐름

오퍼 레터 TC(Total Compensation) 구성

💰 오퍼 구성 항목

· 기본급 (연봉)
· 성과 인센티브 % (연 기본급 대비)
· RSU (주식 부여 총액 + 베스팅 기간)
· 사이닝 보너스
· 재택근무 일수 / 근무지

📊 밴드 연동 기능

· 직군·레벨별 연봉 밴드 자동 조회
· Compa-Ratio 슬라이더 시각화
· contenteditable 인라인 편집
· 자동 저장 (입력 후 5초)
· 인쇄용 오퍼 레터 PDF

역할별 사용법

📌 채용 요청서 작성

채용 → 채용 요청서

사이드바 채용 그룹 → 채용 요청서 목록 → + 새 요청서

직군·레벨·트랙 선택

직군 카드 그리드에서 선택 후 L1~L9 레벨, IC/M 트랙 토글. 연봉 밴드 자동 표시

제출

임시저장 또는 바로 제출. 제출 시 부서장에게 알림 발송

📌 면접관으로 참여

알림 확인

인박스 알림 → 면접 배정 확인

피드백 작성

지원자 상세 → 면접 탭 → 해당 라운드 피드백 작성 (별점 5개 항목 + 합불 추천 + 메모)

📌 파이프라인 관리 (칸반 보드)

채용 → 공고·파이프라인

공고별 지원자 칸반 보드. 드래그앤드롭으로 스테이지 이동

지원자 상세 확인

카드 클릭 → 슬라이드 패널 또는 상세 페이지 (2패널 레이아웃)

면접 라운드 생성

상세 → 면접 탭 → 라운드 추가 → 인터뷰어 배정 (드롭다운 검색)

오퍼 발행

오퍼 탭 → TC 구성 입력 → 오퍼 레터 발행. 상태: draft → sent → accepted/rejected

합격 처리 → 직원 전환

오퍼 수락 후 합격 처리 → 직원 등록 폼에 데이터 자동 프리필 → 저장 시 직원 생성 완료

채용 파이프라인 — 칸반 보드 (드래그앤드롭 스테이지 이동)
주의사항
불합격 처리된 지원자는 파이프라인에서 자동 분리됩니다. 불합격 사유 코드는 채용 컴플라이언스 감사를 위해 영구 보존됩니다.
드래그앤드롭으로 불합격(Disqualified) 열로 이동 시 불합격 사유 입력 모달이 자동으로 팝업됩니다.
