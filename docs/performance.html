성과 관리

목표 설정, 자기·매니저·다면평가, 캘리브레이션, Talent Card, 후계자 계획

기능 개요

성과 관리 모듈은 목표 설정(OKR/KPI)부터 자기·매니저·다면평가, 캘리브레이션, 보상 연동까지 전체 성과 사이클을 지원합니다. Start/Stop/Continue 프레임워크 기반의 다면평가, 9박스 캘리브레이션, Talent Card를 통해 인재를 종합적으로 관리합니다.

성과 사이클 전체 흐름

다면평가(피어리뷰) Start/Stop/Continue 프레임워크

🟢 Continue
계속 유지할 강점
동료가 잘하고 있어서 앞으로도 계속했으면 하는 행동이나 역량을 구체적으로 작성
🔴 Stop
개선이 필요한 행동
팀 협업이나 업무 효율을 저해하는 행동으로 중단 또는 줄였으면 하는 것
🟡 Start
새로 시작했으면 하는 것
동료가 아직 하지 않지만 시작하면 팀과 본인에게 도움이 될 행동이나 접근
익명성 보호: 응답자 3명 미만 시 결과가 비공개 처리됩니다. 소규모 팀에서도 심리적 안전이 보장됩니다.

역할별 사용법

📌 목표 설정 및 자기평가

성과 메뉴 접속

사이드바 성과 클릭 → 현재 평가 주기 확인

목표 등록

+ 목표 추가 → 제목·설명·가중치 입력. 템플릿에서 불러오기 가능

진행률 업데이트

목표별 진행률(%) 정기 업데이트 → 자동 가중 평균 계산

자기평가 작성

각 목표에 1~5점 자기평가 + 코멘트 작성

다면평가 작성

배정된 동료에 대해 SSC(Start/Stop/Continue) 3섹션 작성

등급별 보상 배수(Merit 인상률)가 성과 페이지 상단에 공개됩니다. 목표 설정 시 참고하세요.
성과 관리 — 직원 뷰 (목표 목록, 진행률, 등급 안내)

📌 팀원 평가 및 다면평가 배정

팀원 목표 검토

성과 → 팀원 목록에서 각 팀원의 목표 진행률 확인

매니저 평가 작성

팀원별 1~5점 평가 + 구체적 코멘트. 자기평가 점수와 비교 가능

피어리뷰 배정

성과 → 피어리뷰 배정 탭 → 팀원 간 상호평가 쌍 설정

Talent Card 작성

팀원 프로필 → 성과 탭 → Talent Card 보기 → Retention Risk, Loss Impact 입력

📌 캘리브레이션 및 등급 확정

캘리브레이션 접속

성과 → 캘리브레이션 → 평가 주기 선택

점수 검토 및 등급 조정

자기·피어·매니저 평균 점수 확인. 등급 조정 시 사유 입력 필수 (최대 1단계 하향)

Potential Score 입력

Low/Mid/High 잠재력 평가 → 9박스 Y축 결정

보상 검토 시작

전원 확정 후 보상 검토 시작 버튼 → 보상 관리 Merit 탭으로 자동 이동

캘리브레이션 결과는 직원에게 공개 전까지 비공개입니다. is_shared 토글로 개별 공개 가능합니다.

📌 평가 주기 관리

주기 생성

성과 → 평가 주기 → + 새 주기 (반기/연간/분기 선택)

목표 템플릿 준비

성과 → 목표 템플릿 → 부서·역할별 기본 목표 미리 작성해두면 직원이 바로 적용 가능
