역할별 사용법

📌 급여명세서 확인

내 문서 → 급여명세서

월별 탭으로 급여명세서 목록 확인. 기본급·수당·공제·실수령액 상세 조회

인쇄

페이지 우측 상단 인쇄 버튼 → 브라우저 인쇄(Ctrl+P)로 PDF 저장 가능

📌 증명서 발급 신청

내 문서 → 증명서 발급

발급 허브에서 필요한 증명서 선택

발급 신청

용도 입력 후 발급 신청. HR 승인 완료 시 인쇄 버튼 활성화

📌 전자계약서 서명

인박스 알림 확인

계약서 발행 시 인박스에 CTA 알림. 클릭 → 계약서 보기

계약서 내용 검토 후 서명

하단 전자 서명 버튼 클릭 → 이름 확인 → 서명 완료

📌 계약서 발행

계약서 → 새 계약 발행

직원 선택 → 계약 유형(템플릿) 선택 → 데이터 자동 치환 확인

미리보기 확인 후 발행

우측 미리보기 패널에서 실제 계약서 내용 확인 후 발행. 직원에게 알림 발송

📌 복지포인트 지급

복리후생 → 복지포인트 관리

전체 일괄 지급 또는 개별 지급 선택. 연간 기준 금액 설정 가능

지급 후 확인

직원 복리후생 페이지에서 잔액 및 지급 이력 확인 가능
