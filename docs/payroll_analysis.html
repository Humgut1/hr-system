⚠️ 현재 주요 문제점
🔴 심각 (사용성 저하)
· 급여 생성 전 미리보기 없음 → 실수 후 복구 불가
· 급여명세서 삭제·재생성 기능 없음
· 급여 수정 이력 없음 (현재값만 저장)
· 직원에게 급여 확정 알림 없음
🟠 중요 (기능 부족)
· 일할계산 UI 없음 (코드만 있고 버튼 없음)
· 연봉 기준표 ↔ 실제 급여 미연동
· 일괄 인상 (% 올리기) 기능 없음
· 급여 캘린더 (지급 예정일) 없음
🟡 개선 필요 (UX)
· Admin 급여 페이지가 너무 많은 기능 혼재
· 직원 급여명세서 UI 단조로움
· 복리후생 설정과 급여명세서 연결 불명확
🟢 잘 되는 것
· 4대보험·소득세 자동계산 정확
· 비과세 한도 자동 처리
· 연장·야간·휴일 수당 (§56) 정확
· 퇴직금·미사용연차 자동 계산
· 성과 연동 상여금 (PI/CI)
